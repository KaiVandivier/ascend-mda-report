This is a template for HTML/javascript based reports

You have the jQuery javascript library at your disposal when writing reports. It can be accessed with the usual jQuery or $ symbols.

You can embed javascript through the usual <script type="text/javascript"><script> tag. You can embed CSS styling through the usual <style type="text/css"><style> tag. You should not include <html>, <body> or <head> tags in your report.

You can utilize the DHIS 2 Web API to render meta-data and data in your report. The Web API can be accessed from ../api. For instance, to retrieve the first page of data elements in JSON format you can link to ../api/dataElements.json. To get access to aggregate data you can utilize the analytics resource at ../api/analytics.

HTML-based standard reports can use relative periods and report parameters in the same way as other types of standard reports.
